Homepage Concept 1

View Concept 1

Notes

  • Includes an immersive hero banner
  • Features mobile slide transitions on scroll, inspired by Zara Home. This creates a more editorial, premium feel, but it does require tighter content control. Zara Home relies mainly on imagery for this approach; when text is introduced, copy can quickly become difficult to read if it doesn't fit cleanly within each slide.
  • Category links display text overlaid on imagery.
